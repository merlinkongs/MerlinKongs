# 🎮 MERLINKONGS

MERLINKONGS is a retro-inspired platform adventure built entirely with HTML, CSS, and JavaScript.

## ✨ Features

- 🏰 Multiple adventure levels
- 👑 Boss battles
- 🪙 Coin collection system
- ⚡ Character upgrades
- 🏆 Achievements
- 💾 Save progress
- 📱 Mobile support
- 🎨 Pixel-art graphics

---

## 🎮 Gameplay

Explore dangerous worlds, defeat enemies, collect coins, unlock upgrades, and conquer powerful bosses.

---

## 🕹 Controls

### Desktop
- A / ← : Move Left
- D / → : Move Right
- Space : Jump
- Shift : Sprint

### Mobile
- On-screen controls

- ⏯ Play / Pause Controls
